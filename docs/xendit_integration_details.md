# SafeDrive: Xendit xenPlatform Integration Details

This document outlines the technical architecture and workflow for the Xendit payment integration, replacing the legacy PayMongo system.

## 1. Overview
SafeDrive uses **Xendit xenPlatform** to handle marketplace payments. This allows for automated fund management between the platform (SafeDrive) and car owners.

## 2. Key Features
- **Escrow-style Payments**: Xendit holds the renter's payment in a secure managed account until the rental is complete.
- **Automated Splitting**: The system automatically calculates and distributes the revenue:
    - **90%** to the Car Owner.
    - **10%** to the SafeDrive Platform (Service Fee).
- **Scheduled Payouts**: Funds are released only after the `end_date_time` of a booking has passed.

## 3. The Flow
1. **Booking & Schedule**: 
   - Renter selects a vehicle and sets the `start_date_time` and `end_date_time`.
2. **Checkout**: 
   - Renter pays via a Xendit Invoice generated by the system.
   - Payment methods supported: GCash, Maya, GrabPay, Credit/Debit Cards, and Bank Transfers.
3. **Escrow (Hold)**: 
   - Once paid, Xendit holds the money. The booking status in the system updates to `paid`.
   - The car owner can see that the customer has already paid.
4. **Completion & Distribution**: 
   - When the rental reaches its `end_date_time`, a **Supabase Edge Function** triggers.
   - The function calls the Xendit API to process the split payout.
   - Funds are dispersed to the respective wallets/bank accounts of the car owner and platform owner.

## 4. Technical Architecture
### Database Changes
- **`profiles.xendit_business_id`**: Stores the unique ID of the owner's Xendit Managed Account.
- **`bookings.payment_id`**: Stores the Xendit Invoice ID.
- **`bookings.payout_status`**: Tracks the status of the automated split (`pending`, `completed`).

### Automation (The "Cron" Job)
- **Supabase Edge Function**: `xendit-automated-payout`
- **Schedule**: Runs hourly.
- **Logic**:
    ```sql
    SELECT * FROM bookings 
    WHERE status = 'confirmed' 
      AND payment_status = 'paid' 
      AND payout_status = 'pending' 
      AND end_date_time <= NOW();
    ```

## 5. Setup Requirements
### For Platform Admin
- Xendit API Secret Key (Production & Test).
- xenPlatform enabled in Xendit Dashboard.

### For Car Owners
- Must undergo basic KYC (Identity verification) via the SafeDrive settings page to be registered as a "Managed Account" in Xendit.
